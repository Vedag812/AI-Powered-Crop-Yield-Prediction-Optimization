### **Installation Steps**

#### **1. Clone the Repository**

```bash
git clone https://github.com/yourusername/krishisevak-platform.git
cd krishisevak-platform
```

#### **2. Install Dependencies**

```bash
npm install
# or using yarn
yarn install
```

This will install all required packages including:
- React & React DOM
- Supabase client
- UI component libraries
- Chart libraries
- And 60+ other dependencies

#### **3. Environment Configuration**

Create a `.env` file in the root directory:

```bash
cp env.example .env
```

Edit `.env` with your configuration:

```env
# ================================
# SUPABASE CONFIGURATION (Required)
# ================================
VITE_SUPABASE_URL=https://your-project.supabase.co
VITE_SUPABASE_ANON_KEY=your-anon-key-here

# ================================
# GOOGLE EARTH ENGINE (Optional)
# ================================
VITE_GOOGLE_EARTH_ENGINE_API_KEY=your-gee-api-key
VITE_GEE_PROJECT_ID=your-gee-project-id

# ================================
# WEATHER API (Optional)
# ================================
VITE_WEATHER_API_KEY=your-weather-api-key
VITE_WEATHER_API_PROVIDER=openweathermap

# ================================
# SMS GATEWAY (Optional)
# ================================
VITE_SMS_API_KEY=your-sms-gateway-key
VITE_SMS_SENDER_ID=KRISHI

# ================================
# WHATSAPP BUSINESS API (Optional)
# ================================
VITE_WHATSAPP_API_KEY=your-whatsapp-api-key
VITE_WHATSAPP_PHONE_NUMBER_ID=your-phone-number-id

# ================================
# AADHAAR API (Optional)
# ================================
VITE_AADHAAR_API_KEY=your-aadhaar-api-key
VITE_AADHAAR_API_ENDPOINT=https://api.aadhaar.gov.in

# ================================
# GOOGLE COLAB ML MODELS (Optional)
# ================================
VITE_COLAB_MODEL_ENDPOINT=your-colab-endpoint
VITE_COLAB_API_KEY=your-colab-api-key

# ================================
# APPLICATION SETTINGS
# ================================
VITE_APP_ENV=development
VITE_APP_VERSION=1.0.0
VITE_APP_NAME=KrishiSevak
VITE_DEMO_MODE=true
```

#### **4. Supabase Setup** (Essential)

1. **Create a Supabase Account**
   - Visit [supabase.com](https://supabase.com)
   - Sign up for a free account

2. **Create a New Project**
   - Click "New Project"
   - Project Name: `krishisevak`
   - Database Password: Choose a strong password
   - Region: Select closest to your target users (Asia South for India)

3. **Get API Credentials**
   - Navigate to Settings â†’ API
   - Copy `Project URL` â†’ Add to `.env` as `VITE_SUPABASE_URL`
   - Copy `anon public` key â†’ Add to `.env` as `VITE_SUPABASE_ANON_KEY`

4. **Configure Authentication**
   - Go to Authentication â†’ Providers
   - Enable Email provider
   - Configure settings as needed

5. **Database Schema** (Optional - for full features)
   - Navigate to SQL Editor
   - Run the schema creation scripts (see `supabase/schema.sql`)

#### **5. Start Development Server**

```bash
npm run dev
```

The application will start at `http://localhost:5173`

#### **6. Build for Production**

```bash
npm run build
```

This creates an optimized production build in the `dist/` folder.
